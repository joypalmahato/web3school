## Recommended Pick

**Use Variation B** ("Stop Guessing. Start Building.") as the default.

**Why:**
- Problem-aware audiences convert faster — naming the pain creates instant recognition
- "Stop / Start" structure is action-oriented, creates momentum
- Shortest headline of the three — maximum punch at display size
- Works well at 72px+ where every word must earn its space
- Pairs naturally with the proof bar stats below

**Fallback:** Run A/B test between Variation B (problem-led) and Variation A (benefit-led). Identity copy (C) works better in retargeting and email, not cold landing traffic.

---

## Copy Principles Applied

| Principle | How It's Applied |
|---|---|
| Specificity sells | "90 days", "10 minutes", "50+ roles" — concrete numbers, not vague promises |
| One CTA per viewport | Primary CTA is always "Start Discovery" — secondary is ghost style, lower visual weight |
| Microcopy reduces friction | "Free forever. No credit card." removes the #1 objection before it forms |
| Active voice only | "AI maps", "you learn", "start building" — no passive constructions |
| No jargon in hero | Save "DeFi", "smart contracts" for deeper sections — hero speaks to beginners |
